# haha

> A Vue.js project about Tencent

## Build Setup

``` bash
# 安装依赖
npm install

# 开发环境  监听localhost:8080  自动刷新页面
npm run dev

# 最后上线前build生成dist目录  对js压缩,合并,css压缩,文件名md5等优化操作
npm run build
```
